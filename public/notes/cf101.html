3. HISTORY OF COMPUTER

The history of automatic data processing began with Charles Babbage's attempt to build an automatic mechanical calculator at Cambridge, England in 1830. The entire computer evolution has been divided into a certain number of generations.

  • First Generation (UNIVAC)
  • Second Generation (IBM 1401)
  • Third Generation (HP 9000)
  • Fourth Generation
  • Fifth Generation

First Generation: (1940s-1950s)

First generation computers (1940s-1950s) used vacuum tubes for processing. They were large in size, consumed high power, and had limited programming capabilities. Examples: ENIAC (Electronic Numerical Integrator and Calculator), EDSAC (Electronic Delay Storage Automatic Calculator), EDVAC (Electronic Discrete Variable Automatic Computer), UNIVAC (Universal Automatic Computer).

Used vacuum tubes and consumed high power.

  • Large size
  • Machine language
  • Generated heat

Second Generation

Second generation computers used transistors instead of vacuum tubes, making them smaller, faster and more reliable. These computers consumed less power, produced less heat and had improved memory with magnetic core storage. Example: IBM 1401, IBM 7090

Used transistors instead of vacuum tubes.

  • Smaller size
  • Less heat
  • More reliable

Third Generation

The third generation computers replaced transistors with Integrated Circuits known popularly as chips. From Small Scale Integrated (SSI) circuits which had about 10 transistors per chip, technology developed to Medium Scale Integrated (MSI) circuits with 100 transistors per chip. Examples: IBM 360 Series.

Integrated circuits replaced transistors.

  • Faster
  • Reliable
  • Low power

2) Memory Unit Part

Any storage unit of a computer is known as memory. It is divided into:

  • Primary Memory (Main Memory)
  • Secondary Memory (Storage)
  • Cache Memory

i) Primary Memory

Primary memory is the memory directly accessed by the CPU. It stores data and instructions that the computer is currently using. The primary memory unit is classified into the following:


(a) RAM (Random Access Memory)

RAM is a type of primary memory used by a computer to store data and programs temporarily while they are being used. RAM is a volatile memory (data is lost when the computer is turned off).



Types of RAM:

  • SRAM (Static RAM)
  • DRAM (Dynamic RAM)
  • SDRAM (Synchronous DRAM)
  • DDR SDRAM

SRAM: SRAM is a type of RAM that stores data in a static way, meaning it does not need to be refreshed like DRAM (Dynamic RAM).


DRAM: DRAM is the most common type of main memory (RAM) used in computing devices. It stores each bit of data in a tiny capacitor, which needs to be constantly refreshed to retain the data.


SDRAM: It is a type of DRAM that is synchronized with the system clock - meaning it works in step with the CPU to improve performance and efficiency.


DDR SDRAM: An advanced type of SDRAM that can transfer data twice per clock cycle - once on the rising edge and once on the falling edge of the clock signal. This makes it twice as fast as regular SDRAM.


Versions:

  • DDR1 - First Version
  • DDR2 - Better Speed
  • DDR3 - Higher Speed
  • DDR4 - Faster Performance
  • DDR5 - Latest Technology


(b) ROM (Read Only Memory)

ROM is a non-volatile memory used in computers and electronic devices. It permanently stores data that cannot be easily changed or erased. Unlike RAM, the data in ROM stays even when the power is turned off.



Types of ROM:

  • PROM
  • EPROM
  • EEPROM

PROM: PROM is a type of memory chip that is blank when made and can be programmed only once by the user. After programming, the data cannot be changed or erased.


EPROM: EPROM is a type of ROM that can be erased and reused. To erase the data, it needs to be exposed to ultraviolet (UV) light.


EEPROM: EEPROM is a type of ROM that can be erased and rewritten using electricity, without removing the chip from the computer or device.



ii) Secondary Memory

Secondary memory, also called Auxiliary memory, is used to store data permanently or for long-term storage. It's non-volatile memory; it retains data even when the power is turned off. It is slower than primary memory (RAM) but provides much longer storage capacity.


Types of Secondary Memory:

  • Hard Disk Drive (HDD)
  • Optical Storage
  • SSD (Solid State Drive)
  • USB Flash Drive
  • SD Card

Hard Disk Drive (HDD)

A Hard Disk Drive (HDD) is a data storage device used in computers and other electronic devices. It stores digital information using magnetic storage on one or more rotating platters with head write/read heads that move across the surface to access data.



Advantages:

  • Large storage capacity
  • Cost effective
  • Long lifespan

Disadvantages:

  • Slower than SSD
  • Mechanical failure possible
  • Heavy and noisy


Optical Storage

Optical storage is a type of data storage that uses laser technology to read and write data on a disc surface. It stores data in the form of tiny pits and lands on a reflective surface, typically made of plastic and metal.



Common Types:

  • CD
  • DVD
  • Blu-ray Disc
  • CD-ROM
  • CD-R
  • CD-RW

Advantages:

  • Cheap media
  • Portable
  • Good for archiving

Disadvantages:

  • Slow speed
  • Limited storage
  • Can get scratched


SSD (Solid State Drive)

An SSD is a high-speed storage device that uses flash memory to store data. Unlike traditional HDDs, SSDs have no moving parts, making them faster, more reliable and quieter.




Types of SSD


Type Interface Speed Range Common Use
SATA SSD SATA III ~ 500 MB/s Budget laptops and desktops
NVMe SSD PCIe 1500-7000+ MB/s Gaming, content creation, fast boot
M.2 SSD M.2 Slot SATA or NVMe Slim laptops, modern PCs
U.2 / U.3 SSD PCIe / SAS Enterprise speeds Data centers, servers
External SSD USB-C / USB-A Up to ~1000 MB/s Portable storage




Advantages of SSD

  • Faster data access and boot time
  • No moving parts - more durable
  • Consumes less power
  • Generates less heat
  • Silent operation
  • Lightweight and compact
  • Faster file transfer
  • More reliable over time

Disadvantages of SSD

  • Higher cost per GB
  • Limited write cycles
  • Lower storage capacity
  • Difficult data recovery
  • Performance may degrade


USB Flash Drive

A USB Flash Drive (also known as a pen drive, thumb drive, or memory stick) is a portable solid-state storage device that connects to computers and other devices via a USB port. It uses flash memory to store data.



Advantages:

  • Highly portable
  • Easy plug-and-play use
  • No moving parts
  • Reusable many times
  • Compatible with many devices
  • Low power consumption

Disadvantages:

  • Easily lost
  • Limited write cycles
  • Can carry viruses
  • Slower than SSDs
  • Lower storage capacity


SD Card (Secure Digital Card)

SD Drive (Secure Digital Card): An SD card is a small, portable solid-state storage device that uses flash memory to store data. It is widely used in smartphones, cameras, tablets, drones, and other portable electronics.




Cache Memory

Cache memory is a small, high-speed type of volatile memory that sits between the CPU and main memory (RAM). It stores frequently accessed data and instructions to speed up processing.

The brain of computer.

Functions of CPU:

Types of CPU by cores:

Output Unit

The output unit is the part of the computer that converts processed data (in binary form) from the CPU into a human-readable format such as text, sound, or images and sends it to output devices. Example:- Monitors, Printer, Speakers, Projector, Headphone, plotter.





Software

Software is a collection of programs, instructions, or data used to operate computers and perform specific tasks. It tells the hardware what to do and how to do it.


Types of Software:

  • System Software: Windows, Linux, Android, MacOS
  • Application Software: MS Word, Google Chrome


Hardware

Hardware refers to the physical parts of a computer that you can see and touch. It includes all the mechanical and electronic components that make up a computer system.


  • Input Devices
  • Output Devices
  • Storage Devices
  • Processing Devices (CPU)


GPU (Graphics Processing Unit)

A GPU is a specialized electronic chip designed to handle graphics and visual processing. It was originally built for rendering images, videos, and 3D models in real-time - but now it's also used in AI, gaming, data science, and video editing.



Types of GPU:

  • Integrated GPU
  • Dedicated GPU
  • Workstation GPU
  • External GPU (eGPU)

Common Uses of GPU:

  • Gaming : Renders 3D graphics, effects, and textures smoothly. Enables high FPS (frames per second) gameplay.
  • Video Editing: Accelerates video rendering and playback. Speeds up effects and transitions in editing software.
  • Graphic Design: Helps with high-resolution image editing. Speeds up filters, layers, and 3D animation rendering.
  • Artificial Intelligence & Machine Learning: Trains deep learning models much faster than CPU. Used in TensorFlow, PyTorch, etc.
  • 3D Modeling and CAD Applications: Renders 3D objects, architectural plans, and simulations. Used in AutoCAD, Blender, SolidWorks.
  • Watching High-Resolution Videos (4K/8K): GPU decodes HD/4K videos for smooth playback. Used in YouTube, Netflix, VLC media player.
